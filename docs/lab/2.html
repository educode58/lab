L1 - Configurarea Visual Studio Code cu GitHub

Ce este Visual Studio Code?​

Prezentare VS Code

Visual Studio Code este un program gratuit care te ajută să scrii cod pentru site-uri web, jocuri sau aplicații. Este ca un caiet magic unde poți scrie și testa idei.

Visual Studio Code (denumit VS Code ) este un editor de cod sursă dezvoltat de Microsoft pentru Windows, Linux, macOS și browsere web . Caracteristicile includ suport pentru depanare, evidențierea sintaxei, completarea inteligentă a codului, fragmente de cod, refactorizarea codului și controlul versiunilor încorporat cu Git. Utilizatorii pot schimba tema, comenzile rapide de la tastatură și preferințele.

Ce este GitHub?​

Prezentare Github

GitHub este un loc pe internet unde poți salva proiectele tale de cod. Este ca un dulap mare unde poți păstra lucrurile tale și să le împărtășești cu alții. Folosește Git pentru a ține evidența schimbărilor.

https://github.com/

Pasul 1: Instalează Visual Studio Code​

  1. Deschide browser-ul tău (cum ar fi Google Chrome sau Edge).

  2. Mergi la site-ul oficial: code.visualstudio.com.

  3. Apasă pe butonul "Download for Windows". Va descărca un fișier numit "VSCodeUserSetup-x64-1.xx.x.exe" (numărul poate fi diferit).

  4. După descărcare, dă dublu-click pe fișier și urmează instrucțiunile:

    • Acceptă termenii.
    • Alege unde să instaleze (lasă implicit).
    • Apasă "Next" și "Install".

  5. Când se termină, apasă "Finish". VS Code se va deschide automat.

Felicitări! Ai instalat VS Code.

Pasul 2: Instalează Git​

Git este programul care lucrează cu GitHub.

  1. Mergi la git-scm.com.

  2. Apasă "Download for Windows".

  3. Descarcă versiunea pentru 64-bit.

  4. Rulează installer-ul și urmează pașii:

    • Acceptă licența.
    • Alege componentele (lasă implicit).
    • Alege editorul (recomandăm VS Code).
    • Restul lasă implicit și apasă "Install".

  5. Când se termină, deschide Command Prompt (caută "cmd" în meniu Start) și scrie git --version. Dacă vezi un număr, e bine.

Pasul 3: Creează un cont GitHub​

  1. Mergi la github.com.

  2. Apasă "Sign up" și creează un cont gratuit. Alege un nume de utilizator simplu.

  3. Verifică email-ul tău pentru a confirma contul.

Pasul 4: Creează primul Repository​

Important

Un repository este elementul cel mai important al GitHub. Este un loc unde poți stoca codul, fișierele și istoricul modificărilor fiecărui fișier.

Repository-urile pot avea mai mulți colaboratori și pot fi publice, interne sau private.

  1. În dreapta sus, apasă pe + → New repository.

  2. Completează:

    • Repository name: laborator1
    • Description: „Primul meu repository” (opțional)
    • Selectează Public
    • Bifează Add a README file

  3. Apasă Create repository.

Pasul 5: Configurează VS Code cu GitHub​

  1. Deschide VS Code.

  2. Instalează extensia GitHub:

    • Apasă pe iconița Extensions (pătratul cu linii) din stânga.
    • Caută "GitHub" și instalează "GitHub Pull Requests and Issues" de la GitHub.

  3. Conectează-te la GitHub:

    • Apasă Ctrl+Shift+P (sau Cmd+Shift+P pe Mac, dar tu ai Windows).
    • Scrie "GitHub: Sign in" și apasă Enter.
    • Urmează pașii pentru a te conecta cu contul tău GitHub.
